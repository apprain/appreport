Unzip the files after download and follow below steps. 

Method 1: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Click on (Application) tab
    4.	Click on (Install New Component)
    5.	Select appreport.zip file ( Click on the Floppy Disk image on right side of the upload file input box)
    6.	Click on Install Button. 
    7.	After upload, this Appreport component will appear in the list.
        Now click on "Activate" link blow description. 
        
Method 2: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Unzip appreport.zip.
    4.	Copy "appreport" folder in "component" directory in your project
    5.	In Admin panel go to application tab and find appreport component
    6.	Click on Activate Link. 
	
Install Sample Report:
    1. Login Admin panel  then got to Reports Tab
    2. Go to Dictionary of Groups and Create a Group "Sample Report"
    3. Go to Dictionary of reports and click on "Create Report"
        a) Enter Report Name 
        b) Select group name
        c) Click on Save Button
        d) Click on "Edit Code" link
        e)   i. Unzip sample-reports.zip
            ii. Copy code from "sample_report_1.txt" 
           iii. Paste Code in the Editor 
            iv. Click on "Save" button
             v. Once you see the Message "Save Successfully", Click on "Apply" Button to See the Report Output
        f) Do Same for Sample Report 2

Note: While you create new user in admin panel, you have to assign permission to view report by user.
